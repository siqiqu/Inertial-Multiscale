# Inertial-Multiscale
This repository contains an implementation of an inertial forward-backward splitting algorithm for solving monotone inclusion problems. The scheme combines inertial effects, exterior penalization and Tikhonov regularization. The problem setting and the employed algorithm are described in [[1]](#1).
